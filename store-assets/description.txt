SHORT DESCRIPTION (132 chars max):
Shows Claude Code service status in your browser toolbar. See outages, incidents, and 7-day history at a glance. Unofficial.

---

DETAILED DESCRIPTION:

Keep an eye on Claude Code's health without leaving your browser. This unofficial, independent extension is not affiliated with Anthropic.

It puts a live status indicator in your Chrome toolbar — a colored spark icon that shifts from green through yellow, orange, to red based on Claude Code's current status. Hover to see the status text, click to see the full picture.

WHAT YOU GET:
- Live status icon in the toolbar that updates every 30 seconds
- 7-day history bar with gradient colors matching status.claude.com — hover to see outage type, duration, and related incidents
- Accurate outage durations computed from component-level status transitions
- Current status of all Claude services (claude.ai, API, platform, Cowork, Government)
- Recent incident summaries with timestamps and resolution status
- Light and dark theme support (follows your system setting)

HOW IT WORKS:
The extension polls Anthropic's public status page API (status.claude.com) for real-time data. All processing happens locally in your browser. No accounts, no sign-ups, no data collection.

PERMISSIONS:
- Alarms: periodic status checks
- Storage: cache status data locally
- Host access: status.claude.com only
